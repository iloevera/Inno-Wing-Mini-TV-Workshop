<h1> PCB </h1>

This folder contains the files relating to the PCB
connector board from our workshop. Check MiniTVV4Sch.pdf to 
see how all of our components are being connected.

MiniTVV4.step is a 3D model of the PCB.

Materials used in this workshop:
- ESP32
- ILI9488 TFT display
- Rotary encoder
- Self-locking push button
- Custom PCB (See MiniTVV4Sch.pdf)
- Duopoint wires
